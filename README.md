# 💻 Sistema de Orçamentos (C# Console)

Um projeto de aplicação em console desenvolvido para aplicar e praticar conceitos fundamentais de Programação Orientada a Objetos (POO) utilizando C# e o ecossistema .NET.

## 📝 Sobre o Projeto
Este é um sistema simples focado no cadastro e exibição de orçamentos de serviços. O objetivo principal do repositório é consolidar os estudos práticos em C#, focando na estruturação do código, separação de responsabilidades e interações via terminal.

## 🚀 Funcionalidades (Atuais)
* Definição de modelos de dados (Classes de `Cliente` e `Orcamento`).
* Aplicação do conceito de **Composição** na POO (A classe Orçamento contém um objeto Cliente).
* Geração e exibição de um resumo formatado do orçamento no console.

## 🛠️ Tecnologias e Conceitos Utilizados
* **Linguagem:** C#
* **Plataforma:** .NET (Console Application)
* **Conceitos:**
  * POO (Classes, Campos, Objetos)
  * Modificadores de Acesso (`public`, `internal`)
  * Membros de Instância vs Estáticos

## ▶️ Como Executar
1. Clone este repositório para a sua máquina local.
2. Abra o arquivo `sistema-orcamentos-csharp.sln` no Visual Studio.
3. Defina o projeto como "Projeto de Inicialização".
4. Pressione `F5` ou `Ctrl + F5` para compilar e executar o terminal.
